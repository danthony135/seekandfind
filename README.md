# Seek & Find

A free, private space to explore questions about God, Jesus, faith, and the meaning of life. AI-powered conversations modeled after how Jesus talked to people — questions, stories, patience, love.

## Features

- Personal AI conversations about faith, grounded in scripture
- 20 SEO-optimized question landing pages
- 16 languages with auto-detection and RTL support
- Connect form for real-world follow-up at decision moments
- Admin dashboard with analytics and AI-powered insights
- Completely free. No accounts. Conversations stored locally.

## Quick Start

```bash
npm install
cp .env.example .env    # Add your ANTHROPIC_API_KEY
npm run dev             # Frontend dev server
node server.js          # Backend proxy (separate terminal)
```

## Deploy to Railway

1. Push to GitHub
2. Connect repo in Railway dashboard
3. Add environment variable: `ANTHROPIC_API_KEY`
4. Build command: `npm run build`
5. Start command: `npm start`
